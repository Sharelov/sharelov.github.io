Service Providers

Service Providers are located in App\Providers.

Sharelov uses three servide providers:

  • LibreriaProvider is the main provider with all the app's bindings, observers and view composers.

  • LibreriaMacrosProvider is where all the HTML and Form macros live.

  • SubscriptionProvider contains all the bindings related to the subscription service
