Classes that deal with the uploaded assets.

Uploads

See Also

  • Facade: UtilUploads
  • Trait: App\Libreria\Traits\UploadsTrait

namespace: App\Libreria\Assets\Uploads

A class with helpers methods to deal with uploads. Supports uploading files by passing a Model as a parameter.

Upload

Uploads a file by type and inserts it to the database

Params:

  • $field string: The name of the file input field.
  • $type string: File's Model Type. About Content Types.
  • $relation_id integer optional: Id from the model that the image belongs.
  • $options array optional: Array options.
    • max_size: integer: Upload Max Size.
    • path string: Optional image path if different from Model Type's path.
    • additional_info array: Additional info you want to send to the Repository when the create is executed.

Return:

App\Models\Archivo - It Return an Archivo's instance of the uploaded file.

Example:

// import class
use \App\Libreria\Assets\Uploads;
// create instance
$upload_instance = new Uploads;
// upload
$archivo_model = $upload_instance->Upload('upload', 'brands', 14);

getTypePath

Gets a path from a Model Type.

Params:

  • $type string: File's Model Type. About Content Types.

Return:

string - Return the path where a file should be uploaded from a Model Type

Example:

// import class
use \App\Libreria\Assets\Uploads;
// create instance
$upload_instance = new Uploads;
// upload path
$path = $upload_instance->getTypePath('brands');

FileSystem

See Also

  • Facade: UtilFilesystem

namespace: App\Libreria\Assets\FileSystem

A class with helper methods to deal with Laravel's FileSystem. It's mostly used by the Thumbnail class.

getFile

Gets a image string from an image path.

Params:

  • $path string: File path.
  • $disk string optional: FileSystem disk (default is 'uploads')

Return:

string - Return a image string.

Example:

// import class
use \App\Libreria\Assets\FileSystem;
// create instance
$filesystem_instance = new FileSystem;
// image
$image_string = $filesystem_instance->getFile('archivo/posts/123.jpg');

getFileFromModel

Gets a image string from an Archivo elequent model.

Params:

  • $model App\Models\Archivo: An Archivo elequent model instance.
  • $disk string optional: FileSystem disk (default is 'uploads')

Return:

string - Return a image string.

Example:

// import class
use \App\Libreria\Assets\FileSystem;
// create instance
$filesystem_instance = new FileSystem;
// model
$model = Archivo::find(1);
// image
$image_string = $filesystem_instance->getFileFromModel($model);

moveTempFileFromModel

Moves temp type files to another type.

Params:

  • $model App\Models\Archivo: An Archivo elequent model instance with a temp type.
  • $type string: The new Model Type. About Content Types.
  • $relation_id integer: Id from the model that the image belongs.
  • delete_old boolean: Delete temp file. The default is true.
  • save boolean: Saves to database instead of returning the laravel object. The default is true.

Return:

App\Models\Archivo - It Return an Archivo's instance of the uploaded file.

Example:

// import class
use \App\Libreria\Assets\FileSystem;
// create instance
$filesystem_instance = new FileSystem;
// model
$model = Archivo::find(1);
// image
$image_string = $filesystem_instance->moveTempFileFromModel($model, 'tweets', 134);

Params:

  • $type string: The new Model Type. About Content Types.

  • delete_old boolean: Delete temp file. The default is true.
  • save boolean: Saves to database instead of returning the laravel object. The default is true.

Return:

App\Models\Archivo - It Return an Archivo's instance of the uploaded file.

Example:

// import class
use \App\Libreria\Assets\FileSystem;
// create instance
$filesystem_instance = new FileSystem;
// image
$image_string = $filesystem_instance->moveTempFileFromId(143, 'tweets', 134);

Copies an existing file to a new location on the disk.

Params:

  • $old_file_path string: Old file path.
  • $new_file_path string: New file path.

Return:

void

Example:

// use

AssetsUrl

See Also

  • Facade: UtilAssetsUrl

namespace: App\Libreria\Assets\AssetsUrl

Helps creating resized (thumbnails) uploaded images urls. A shortcut to getThumbnailUrlByModel using the option: ['avatar' => true];

getAvatar

Gets the public user profile image that can be accessed outside sharelov (no login needed).

Params:

  • $model App\Models\Archivo: an Archivo elequent model.
  • $params array optional: An array with the thumbnail parameters:
$params = [
    // max width
    'w' => 100,
    // max height
    'h' => 100,
    // crop
    'c' => true,
    // quality
    'q' => 90,
];

Return:

string - Return an image url.

Example:

// import class
use \App\Libreria\Assets\AssetUrls;
// create instance
$assets_helper = new AssetUrls();
// image params
$params = ['w' => 100,'h' => 100];
// model
$model = Archivo::find(1);
// use
$avatar_public_url = $assets_helper->getAvatar(model, params);

getThumbnailUrl

Gets a thumbnail url from an image path.

Params:

  • $image_name string: Image name.
  • $image_path string optional: Image path archivos/brands/
  • $params array optional: An array with the thumbnail parameters:
$params = [
    // max width
    'w' => 100,
    // max height
    'h' => 100,
    // crop
    'c' => true,
    // quality
    'q' => 90,
];
  • $options array optional: An array with the thumbnail options:
// one of the following routes options
$options = [
    // can be accessed outside sharelov (no login needed), with a hashed key
    'protected' => true,
    // can be accessed outside sharelov (no login needed).
    'public' => true,
    // user profile image that can be accessed outside sharelov (no login needed)
    'avatar' => true,
];

Return:

string - Return an image url.

Example:

// import class
use \App\Libreria\Assets\AssetUrls;
// create instance
$assets_helper = new AssetUrls();
// image name
$image_name = '1472581969-HlxnGH88Nb.jpg';
// image path
$image_path = 'archivos/brands/';
// image params
$params = ['w' => 100,'h' => 100];
// use
$avatar_public_url = $assets_helper->getThumbnailUrl($image_name, $image_path, $params);
